Features
Everything a hospital needs
Built for real emergency scenarios — no medical advice, pure asset logistics.
Real-time inventory
Every asset across all departments tracked live. Color-coded bars show stock health at a glance.
Severity-ranked alerts
Alert Agent classifies every shortage as Critical, High, or Medium and logs it to MongoDB instantly.
Auto reallocation
Allocator Agent finds surplus departments and moves assets to critical ones — zero manual steps.
Smart orchestration
One natural language query — the Orchestrator decides which agent handles it automatically.
Daily reports
Report Agent reads the full event log and generates a professional admin summary every day.
MongoDB + Compass
All data stored locally in MongoDB. Watch collections update live in Compass as agents act.

How it works
From query to action in seconds
Type anything in plain English — the system handles the rest.
STEP 01
Staff types a query
"Which assets are critically low?" — plain English, no special commands needed.
STEP 02
Orchestrator routes it
AI decides whether Inventory, Alert, Allocator, or Report agent should handle it.
STEP 03
Agent reads MongoDB
The specialist agent pulls live data from your local database and reasons over it.
STEP 04
Action + log
Response is returned, database is updated if needed, and every action is logged to event_log.
